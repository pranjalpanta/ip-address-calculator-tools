# IP Address Calculator Tools

A modern and responsive IPv4 Address Calculator built with **HTML**, **CSS**, and **JavaScript**. It helps users calculate subnet information from an IPv4 address or CIDR notation instantly. All calculations are performed locally in the browser without requiring a backend server.

## ✨ Features

- IPv4 address validation.
- CIDR notation support (`/0` to `/32`).
- Automatic IP Class detection.
- Public, Private, and Loopback address identification.
- Subnet Mask calculation.
- Network Address calculation.
- Broadcast Address calculation.
- Usable Host Range calculation.
- Total Usable IP Address calculation.
- Binary Network and Host Bit visualization.
- Copy-to-Clipboard support.
- Responsive user interface.
- Fully client-side application.

## ▶️ Run Locally

Clone the repository.

```bash
git clone https://github.com/YOUR-USERNAME/ip-address-calculator-tools.git
```

Go to the project folder.

```bash
cd ip-address-calculator-tools
```

Open `index.html` in your web browser.

Or start a local server.

```bash
python -m http.server 8000
```

Open:

```text
http://localhost:8000
```
